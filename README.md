## ✅ Kako napraviti COMMIT u Visual Studio 2026 

Mini upute za učenike: COMMIT u Visual Studio 2026 (korak po korak)

Otvorite projekt u Visual Studio 2026.

Na vrhu Visual Studija (iznad koda) imaš menije:

File | Edit | View | Git | Project | Build | Debug | …

➡️ Klikni Git
➡️ zatim Git Changes

U prozoru Git Changes provjerite da postoje datoteke pod Changes (to su vaše promjene).

U polje Message upišite kratku poruku (npr. Dodan zadatak 3 ili Ispravak greške).

Kliknite Commit All.

Kliknite Push (ili Sync) da se promjene pošalju na GitHub.

✅ Gotovo — sada je zadatak spremljen i na GitHubu.
