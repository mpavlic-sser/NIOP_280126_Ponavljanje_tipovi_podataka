# Zadatak 1 — Učenik (unos podataka)

Program treba:

- Zatražiti od korisnika unos:
  - imena učenika (string)
  - prezimena učenika (string)
  - razreda koji učenik pohađa (broj od 1 do 4 – tip **byte**)
  - smjera koji učenik pohađa (string)
- Zatražiti unos godina učenika (**int**)
- Provjeriti je li učenik punoljetan (**bool**, 18 ili više godina)
- Nakon unosa podataka, ispisati **jednu rečenicu** koristeći **string interpolaciju**:

```
Učenik Ime Prezime pohađa X. razred smjera Smjer.
```
