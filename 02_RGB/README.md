# Zadatak 2 — RGB

Program treba:

1. Zatražiti od korisnika unos:
   - R (crvena komponenta) – cijeli broj od 0 do 255
   - G (zelena komponenta) – cijeli broj od 0 do 255
   - B (plava komponenta) – cijeli broj od 0 do 255
2. Pretvoriti unesene vrijednosti u tip **byte**
3. Ispisati RGB vrijednost u obliku:

```
RGB(R, G, B)
```
